# emetringapi

API Server for eMetering

## Getting Started

### Prerequisites

NodeJS v9.5.0
NPM 5.6.0

### Installing

1. Pull or Download the files

2. NPM install all packages
```
npm install
```

3. Start Server

```
node index.js
```
